# Examensarbete ScoreHub

## Projektbeskrivning
En enkel Java Spring Boot-applikation som hanterar fotbollslag (teams). Använder Spring Data JPA och PostgreSQL.

## Work in Progress
Detta dokument uppdateras kontinuerligt allteftersom projektet utvecklas.
